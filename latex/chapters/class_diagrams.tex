\chapter{Class Diagrams}\label{ch:class_diagrams}
For each diagram are indicated the classes, variables and methods useful for the action described.\\
\begin{enumerate}
    \item~\fullref{fig:cls_simple_alert_add} Describes the addition of an alert to the system.
    \item~\fullref{fig:cls_simple_alert_edit} Describes the modification of an alert.
    \item~\fullref{fig:cls_simple_alert_delete} Describes the deletion of an alert.
    \item~\fullref{fig:cls_export_alert} Describes how to export a JSON alert file to the file system.
    \item~\fullref{fig:cls_import_alert} Describes how to import a JSON alert file from the file system.
    \item~\fullref{fig:cls_silent_mode} Describes switching to silent mode.
    \item~\fullref{fig:cls_simple_alert_notification} Describes the mechanism for triggering and displaying notifications.
    \item~\fullref{fig:cls_exit} Describes the closing of the program.
\end{enumerate}
