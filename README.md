## Installation 

### Requirements

* python => 3.6
* pygame => 1.9.3
* PyQt5 => 5.8.2
* sip => 4.19.2
* tinydb => 3.2.2

```bash
> pip3 install pygame pyqt5 sip tinydb
```

```bash
> git clone https://github.com/BFH-BTI7301-project1/ClockAlarm.git
```

## Usage

```bash
> python3 bin/clockalarm

```
